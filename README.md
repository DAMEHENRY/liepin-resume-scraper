本项目旨在提供一个自动化的环境设置解决方案，主要通过Python脚本实现。它包含跨平台的设置指南（Mac和Windows），并管理所有必要的依赖项，以简化新环境的配置过程。

python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
